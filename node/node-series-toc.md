# Mục Lục Series Node.js Cho Người Mới Học Backend

## Giới Thiệu
Series Node.js này bao gồm 7 files hướng dẫn chi tiết từ cơ bản đến nâng cao, giúp người mới hiểu và xây dựng backend với Node.js, Express, và các công nghệ liên quan. Mỗi file có TOC và Tổng Kết để dễ theo dõi.

## Danh Sách Files

- [00: How Websites Work](./00-how-websites-work.md) - Giải thích phía sau website, vai trò của backend, JavaScript, Node.js, và Express.
- [01: Node.js vs Other Backends](./01-node-vs-other-backends.md) - So sánh Node.js với Python (Django/Flask), PHP (Laravel), Ruby (Rails), Java (Spring).
- [02: Node.js Basics Guide](./02-node-basics-guide.md) - Cơ bản Node.js: modules, file system, HTTP server, asynchronous programming, events.
- [03: Express.js Guide](./03-express-guide.md) - Hướng dẫn Express: routing, middleware, CRUD operations, error handling, authentication.
- [04: Node.js Databases Guide](./04-node-databases-guide.md) - Kết nối Node.js với databases: MongoDB (NoSQL), PostgreSQL (SQL), ORMs, authentication.
- [05: Node.js Operations Types](./05-node-operations-types.md) - Các loại operations: synchronous/asynchronous, HTTP, file system, database, events, streams, child processes, timers, error handling.
- [06: Advanced Node.js Guide](./06-advanced-node-guide.md) - Nâng cao: streams, clusters, performance tuning, security, authentication, testing, deployment, monitoring, microservices, real-time apps.

## Lộ Trình Học
1. Bắt đầu từ file 00 để hiểu backend cơ bản.
2. Học Node.js cơ bản (01-02).
3. Xây dựng apps với Express và databases (03-04).
4. Khám phá operations và nâng cao (05-06).
5. Thực hành với projects thực tế.

Chúc học vui với series Node.js!
