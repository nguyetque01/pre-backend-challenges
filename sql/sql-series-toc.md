# Mục Lục Series SQL Cho Người Mới Học Backend

## Giới Thiệu
Series SQL này bao gồm 6 files hướng dẫn chi tiết về databases, từ so sánh SQL vs NoSQL đến queries nâng cao. Phù hợp cho người mới học backend, tập trung vào MongoDB (NoSQL), PostgreSQL (SQL), và các kỹ thuật truy vấn.

## Danh Sách Files

- [01: SQL vs NoSQL Comparison](./01-sql-vs-nosql-comparison.md) - So sánh SQL (quan hệ) vs NoSQL (không quan hệ), khi nào dùng cái nào.
- [02: NoSQL Backend Guide](./02-nosql-backend-guide.md) - Hướng dẫn MongoDB: CRUD operations, querying, aggregation, indexing, data modeling.
- [03: SQL Basics Guide](./03-sql-basics-guide.md) - Cơ bản SQL: DDL (thiết kế), DML (thao tác), DQL (truy vấn), DCL (bảo mật), TCL (transactions).
- [04: PostgreSQL Guide](./04-postgres-guide.md) - Hướng dẫn PostgreSQL: cài đặt, DDL với JSON/ARRAY, CRUD nâng cao, querying, bảo mật, hiệu suất.
- [05: SQL Queries Types](./05-sql-queries-types.md) - Các loại queries SQL: DDL/DML/DQL/DCL/TCL, nâng cao với indexing, views, triggers, API integration.
- [06: Advanced SQL SELECT Guide](./06-advanced-sql-select-guide.md) - SELECT nâng cao: JOIN, WHERE, functions, subqueries, CTEs, recursive queries, JSON, metadata.

## Lộ Trình Học
1. Bắt đầu với so sánh SQL vs NoSQL (01).
2. Học NoSQL với MongoDB (02).
3. Cơ bản SQL và PostgreSQL (03-04).
4. Các loại queries và nâng cao (05-06).
5. Thực hành với database thực tế.
